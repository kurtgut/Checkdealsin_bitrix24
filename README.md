Описание программы проверки интернет-сделок в Bitrix24
# Версия 0.1.0
Программа предназначена для проверки интернет-сделок, которые находятся в стадии 'Успешная'. 
Она проверяет наличие файла '1С Накладная' и если файла нет, возвращает сделку на стадию 'Получена оплата'.
+ Добавлена программа расписание для автоматического запуска программы на ПК по расписанию.
+ Она запускается по расписанию каждый час (на 1‑й минуте) и каждые полчаса (на 30‑й минуте), чтобы убедиться, что у сделок прикреплён файл «1С Накладная». Если файла нет, сделка возвращается на стадию «Получена оплата».

Основные шаги работы
1.	1. Настройка логирования.
2.	2. Получение списка сделок за последние 7 дней.
3.	3. Проверка наличия файла в поле сделки.
4.	4. Формирование списка сделок без файла.
5.	5. Перемещение таких сделок на стадию 'Получена оплата'.

Используемые технологии
- Python
- requests (для работы с API)
- loguru (для логирования)
- Bitrix24 API

Итог
Программа автоматизирует контроль качества сделок в CRM Bitrix24, обеспечивая корректность данных и возвращая сделки на доработку при отсутствии накладной.
